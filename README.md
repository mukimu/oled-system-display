# oled-system-display
Just some simple YAML to pull Home Assistant sensor data into an OLED display.

ESP32-WROOM-32 Pinout:

Display:
```
VCC: 3V3
GND: GND
SCL: D22
SDA: D21
```

Rotary Encoder:
```
GND: GND
+: 3V3
SW:
DT: D18
CLK: D19
```
